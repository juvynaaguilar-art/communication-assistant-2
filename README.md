# Collaborative Clarity Coach
[cite_start]This bot is a training tool for **Case Managers and Paraprofessionals** based on UDL frameworks[cite: 4].

## Features
* [cite_start]**Barrier Analysis**: Identifies 'Frequent Interruptions' and 'Conflicting Instructional Cues'[cite: 8, 79].
* [cite_start]**SBI Feedback Coaching**: Helps adults move from subjective 'Blur Words' to objective data[cite: 103, 112].
* [cite_start]**Toolkit Recommendations**: Suggests 'Shared Digital Organizers' and 'Visual Schedules'[cite: 19, 20].

## How to use
1. Open `index.html` in your browser.
2. Enter a classroom scenario to receive coaching on adult-to-adult communication.
